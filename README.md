# Open Draw - Make learning engaging
A libre drawing application for mobile, tablet, and ifp


### Developer Guide

Technology Used :
* Kotlin, XML, Jetpack Compose
* RoomDb, Compose Canvas

Description :  
* Android Single Activity + MVVM + Clean Architecture
* Modularized by feature, current offering (~home, `board)
* Migrating to newer api with compose with backward compatibility support with xml

Deliverables : 
* Home & Board Prototype with brush, eraser and highlighter support

In Progress :
* Undo, redo
* Import, export
* Package optimization
* other tooltips implementation
